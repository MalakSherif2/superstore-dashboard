# Superstore Dashboard 📊

## 📌 Overview

This dashboard analyzes sales and profitability for a retail superstore.

## 📊 Key Metrics

* Total Sales: $2.30M
* Total Profit: $286.40K
* Profit Margin: 12.47%

## 📈 Insights

* West region has the highest profit margin
* Tables category is causing losses
* Standard shipping is the most used

## 🛠 Tools Used

* Power BI
